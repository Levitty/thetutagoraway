Facebook Auto-Posting Setup

Get your Page Access Token in 5 minutes

1

Create a Meta Developer App

  1. Go to developers.facebook.com/apps
  2. Click "Create App"
  3. Select "Other" for use case, then "Business" as app type
  4. Name it "Tutagora Posts" and click Create
2

Get Your Page ID

  1. Go to your Facebook Tutagora Page
  2. Click "About" on the left sidebar
  3. Scroll down — you'll see "Page ID" at the bottom
  4. Copy that number
3

Generate a Page Access Token

  1. Go to the Graph API Explorer
  2. In the top-right dropdown, select your "Tutagora Posts" app
  3. Click "Generate Access Token"
  4. When prompted, add these permissions:
pages_manage_posts pages_read_engagement pages_show_list
  1. After granting permissions, click the "User or Page" dropdown (next to the token field)
  2. Switch to your Tutagora Page — this gives you a Page Access Token
  3. Copy the token
This token expires in ~1 hour. Step 4 makes it permanent.
4

Make It a Long-Lived Token

  1. Go to the Access Token Debugger
  2. Paste your Page Access Token and click "Debug"
  3. Click "Extend Access Token" at the bottom
  4. Copy the new long-lived token (lasts ~60 days)
For a never-expiring token: In the Graph API Explorer, use the long-lived user token to request:
GET /me/accounts
The Page token returned here is permanent.
5

Send Me the Details

  1. Your Page ID (a number like 123456789)
  2. Your Page Access Token (a long string starting with EAA...)

I'll plug them in and your auto-posting will be live!
